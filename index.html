Bistro FDI

Este proyecto consiste en el desarrollo de una página web para ayudar a la gestión de un restaurante.
A la página podrán acceder los clientes, para realizar pedidos.
Los cocineros y camareros preparan y entregan los pedidos.
Gerente administra el inventario y supervisa los servicios que proporciona el restaurante.
